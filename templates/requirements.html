Bot Submission Requirements

Before submitting your bot, make sure it follows all of our rules/guidelines.

  • The bot must be online when we are testing it.
  • Any NSFW commands can not be executed outside of an NSFW channel. This includes any endpoints with a possibility of a NSFW response e.g. nekos.life
  • The bot description must be relevant to the bot, without links to social media platforms.
  • The bot description must be readable without special characters.(𝔱𝔢𝔵𝔱, t̵̩̿ȅ̴̋ẋ̸͚t̶̅͝)
  • Bot can not reply to other bots
  • Bot must have a majority of its commands working, and all important commands (help, info) must work without bugs.
  • Bot must be public and with out requiring a code grant.
  • Bot can not be full pro/premium, it must have some free functionality.
  • Bot must not be a unedited fork.
  • Bot does not need administrator permissions.
  • Bot must not use the `/` prefix without application commands.
  • Usage of a common prefix like `!` is not recommended.

The rules are subject to change at anytime, with or without notice.
